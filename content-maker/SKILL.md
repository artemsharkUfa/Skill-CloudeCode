# Content Maker для Neironka_live

Скилл для создания постов Telegram-канала @Neironka_live (AI, технологии, бизнес, личный опыт).

## Авторский стиль

Прочитать `references/style-guide.md` перед каждым постом. Это обязательно.

## КРИТИЧНЫЕ ПРАВИЛА

1. **Максимум 1 эмодзи на весь пост.** Можно ноль. Никогда больше одного.
2. **Никаких длинных тире (--).** Использовать короткое тире (-) или перестроить предложение.
3. **Тон: разговорный, ироничный, с позицией.** Не нейтральный новостник.
4. **Своё мнение обязательно.** Каждый пост содержит авторскую позицию или вывод.

## Workflow

### Шаг 1: Определить формат

Спросить у пользователя или определить по теме:

| Формат | Длина | Когда |
|--------|-------|-------|
| Tech-обзор | 150-400 слов | Новый инструмент, фича, продукт |
| Аналитика | 300-800 слов | Отчёт, законопроект, рынок |
| Личная история | 400-1000 слов | Опыт, рефлексия, кейс |
| Короткий промо | 50-100 слов | Скидка, ссылка, анонс |
| Мнение/колонка | 200-500 слов | Событие, тренд, критика |

### Шаг 2: Исследование

Если тема требует фактов:
- Веб-поиск актуальной информации
- Проверка источников (официальные сайты, документация)
- Сбор цифр и данных

### Шаг 3: Написание

Структура поста (адаптировать под формат):
1. **Зацепка** - первые 2-3 предложения, которые втягивают. Без банальностей.
2. **Суть** - факты, разбор, описание. Конкретика, без воды.
3. **Авторская позиция** - что я думаю, почему это важно/неважно.
4. **Закрытие** - вывод, вопрос к аудитории, или ироничная ремарка.

### Шаг 4: Проверка по чеклисту

- [ ] Эмодзи: 0 или 1 на весь пост
- [ ] Нет длинных тире
- [ ] Есть авторская позиция
- [ ] Первые два предложения цепляют
- [ ] Нет канцелярита и воды
- [ ] Нет фраз: "в современном мире", "не секрет что", "стоит отметить"
- [ ] Форматирование: жирный для ключевых слов, списки где уместно
- [ ] Длина соответствует формату

### Шаг 5: Выдача

Пост выдавать готовым к копированию в Telegram. Без обёрток, без пояснений сверху.

## Обработка особых случаев

**Контент-план:**
Создать таблицу на неделю/месяц с темами, форматами, датами. Использовать xlsx-maker для файла.

**Серия постов:**
Разбить тему на 2-3 поста с логической связкой. Каждый пост самодостаточный.

**Пост на основе ссылки:**
Прочитать материал через web_fetch, извлечь суть, написать в авторском стиле.
