# PPTX Maker

Скилл для создания PowerPoint презентаций через python-pptx с системой корпоративных тем.

## Workflow

### Шаг 1: Сбор требований

Узнать у пользователя (не больше 2-3 вопросов за раз):
1. Тема/содержание презентации
2. Тип: питч-дек, аналитический отчёт, обучение, корпоративная
3. Количество слайдов (если не указано — предложить оптимальное)
4. Для какой компании/в каком стиле? (если не указано — спросить)

### Шаг 2: Выбор темы

Прочитать `references/themes/` и найти подходящую тему.

**Если тема найдена** — использовать цвета, шрифты, стиль из файла.
**Если темы нет** — спросить у пользователя:
- Основные цвета (фон, акцент, текст)
- Шрифты (или оставить дефолтные)
- Стиль (формальный/современный/минималистичный)
- Логотип (путь к файлу, если есть)

Затем создать новый файл темы в `references/themes/` для будущего использования.

### Шаг 3: Планирование структуры

Определить слайды. Типы слайдов — см. `references/slide-layouts.md`.

Структуру представить пользователю для подтверждения:
```
1. Титульный слайд
2. Повестка / Содержание
3-N. Контентные слайды
N+1. Заключение / CTA
```

### Шаг 4: Генерация

Запустить скрипт:
```bash
python3 ~/.claude/skills/pptx-maker/scripts/create_pptx.py \
  --output "~/outputs/presentation-name.pptx" \
  --theme "fintech-ai" \
  --structure "structure.json"
```

Или собрать презентацию напрямую через python-pptx в inline-коде, используя параметры темы.

**Принципы генерации:**
- Каждый слайд — одна ключевая мысль
- Текст на слайде: максимум 6 строк по 6 слов (правило 6x6)
- Заголовки короткие и конкретные
- Графики/таблицы вместо текста где возможно
- Единообразное форматирование на всех слайдах

### Шаг 5: Валидация

Проверить:
- [ ] Файл создан и открывается
- [ ] Количество слайдов соответствует плану
- [ ] Тема применена (цвета, шрифты)
- [ ] Нет пустых слайдов
- [ ] Текст не обрезается

### Шаг 6: Выдача

Сообщить путь к файлу и краткое описание содержимого.
Файлы сохраняются в `~/outputs/`.

## Работа с данными из Excel

Если пользователь просит создать презентацию на основе Excel-файла:
1. Прочитать .xlsx через openpyxl
2. Извлечь данные, таблицы, числа
3. Визуализировать в слайдах: таблицы → Table shape, числа → Chart shape
4. Применить тему компании

## Обработка особых случаев

**Пользователь даёт только тему без деталей:**
Предложить структуру на основе типа презентации, показать для подтверждения.

**Пользователь хочет редактировать существующую презентацию:**
Прочитать .pptx через python-pptx, показать текущую структуру, внести изменения.

**Нужна презентация на основе документа/текста:**
Извлечь ключевые тезисы, структурировать по слайдам, подтвердить у пользователя.

## Частые ошибки — НЕ допускать

1. Слайды-портянки с кучей текста
2. Несогласованные шрифты/цвета между слайдами
3. Отсутствие визуальной иерархии
4. Логотип растянут или пикселизирован
5. Графики без подписей осей
